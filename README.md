# HR Analytics SQL Project

## Project Overview

This project demonstrates SQL skills by analyzing HR data such as employee performance, promotions, attendance, and salary insights. The goal is to extract meaningful business insights using SQL queries.

## Database Tables

The project uses the following tables:

* **employees** – Employee details, salary, department, performance rating
* **departments** – Department information
* **attendance** – Employee attendance records
* **promotions** – Promotion history of employees

## Key SQL Analysis

### 1. Department Employee Count

Find the number of employees in each department.

### 2. Average Salary per Department

Analyze salary distribution across departments.

### 3. Highest Paid Employee in Each Department

Identify the top earning employee per department.

### 4. Promotion Eligibility List

Find top performers eligible for promotion using window functions.

### 5. Employees Without Promotion

Identify employees who have never been promoted.

### 6. Top 2 Employees by Salary in Each Department

Use `RANK()` window function to determine top earners.

### 7. Employees Earning Above Department Average

Use correlated subqueries to identify high performers.

### 8. Attendance Analysis

Calculate present days and absence counts for employees.

### 9. Employee Attrition Risk

Detect employees with low performance and high absence.

## Skills Demonstrated

* SQL Joins
* Aggregations (SUM, AVG, COUNT)
* Window Functions (RANK, PARTITION BY)
* Subqueries and Correlated Subqueries
* Data Analysis for HR insights
